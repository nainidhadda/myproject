# Career Intelligence Engine

The Intelligence Engine is responsible for converting user data into personalized career insights.

Every recommendation generated by the platform should answer a real user question and provide supporting evidence.

| Intelligence Module | User Question | Evidence Used | Recommendation | Explainability |
|---------------------|--------------|---------------|----------------|----------------|
| Resume Intelligence | Which resume should I use? | Resume versions, JD, previous applications | Recommend best resume | Show previous performance |
| Resume Intelligence | What should I remove from my resume? | Resume, target role | Remove irrelevant content | Explain relevance |
| Project Intelligence | What project should I build next? | Skills, GitHub, target role, saved JDs | Recommend project | Show skill gap and demand |
| Project Intelligence | Which of my projects adds the most value? | Resume, GitHub, projects | Rank projects | Explain ranking |
| Skill Intelligence | What skill should I learn next? | Skill graph, market demand | Prioritize learning | Show ROI |
| Skill Intelligence | Which skills are missing? | Resume, GitHub, JDs | Recommend missing skills | Show frequency in JDs |
| GitHub Intelligence | Does my GitHub support my target role? | GitHub profile | Suggest improvements | Show missing evidence |
| LinkedIn Intelligence | How can I improve my LinkedIn profile? | LinkedIn profile | Recommend updates | Explain impact |
| Application Intelligence | Why am I getting rejected? | Applications, outcomes | Identify patterns | Show supporting evidence |
| Career Intelligence | Am I improving? | Historical data | Show trajectory | Compare previous months |
| Career Intelligence | Which companies should I target? | Skills, projects, saved JDs | Recommend companies | Show match reasoning |
| Career Intelligence | What is the highest impact action I can take today? | Entire career profile | Prioritize one action | Explain expected impact |
