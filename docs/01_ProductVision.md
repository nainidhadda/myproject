# Product Vision

## Vision Statement

Build an AI-powered Career Intelligence Platform that continuously understands a student's professional journey and provides personalized recommendations to improve career decisions over time.

## Problem Statement

Students preparing for internships and entry-level jobs rely on multiple disconnected platforms throughout their job search.

They use:
- Job portals to discover opportunities.
- AI chatbots to tailor resumes.
- GitHub to showcase projects.
- LinkedIn as a professional profile.
- Spreadsheets to track applications.

These tools store information but do not learn from it.

As a result, students repeatedly make career decisions without understanding which resumes perform better, which projects strengthen their profile, which skills provide the highest impact, or how they are improving over time.

## Solution

The platform collects evidence from different parts of a student's career journey and transforms it into personalized career intelligence.

Instead of giving generic advice, every recommendation is based on the user's own profile, goals, projects, skills, GitHub activity, job descriptions, and application history.

## Target Users

- College students
- Internship seekers
- Fresh graduates
- Entry-level software engineers

## Core Philosophy

The platform does not help users apply to more jobs.

It helps them make better career decisions.

## Design Principles

1. Every recommendation must be backed by evidence.
2. Recommendations should improve as more data becomes available.
3. The platform should explain every recommendation.
4. Intelligence should be personalized, not generic.
5. AI should support decision-making rather than replace it.
